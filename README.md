# monthlet releases

[monthlet](https://monthlet.ai) の配布物（DMG / 自動アップデートフィード）を公開するリポジトリです。ソースコードは含まれません。

- 最新版のダウンロード: [Releases](https://github.com/t27d/monthlet-releases/releases/latest) から `monthlet.dmg`（macOS 13+ / Apple Silicon 専用）
- `*.zip` と `latest-mac.yml` はアプリ内自動アップデート（electron-updater）用のファイルです
